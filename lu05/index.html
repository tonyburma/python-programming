Topic 1: Regular Expressions (re)

Used for searching, matching, and manipulating text patterns.


import re
text = "Contact us at info@school.edu"
match = re.search(r'[\w\.-]+@[\w\.-]+', text)
print(match.group()) # info@school.edu

Decoding RegEx Symbols

RegEx uses Metacharacters to represent types of data:

Symbol Meaning Example
\w Word character (a-z, A-Z, 0-9, _) \w+ matches "Python"
\d Digit (0-9) \d{3} matches "555"
\. A literal dot (the \ "escapes" it) \.com matches ".com"
+ One or more of the previous a+ matches "a" or "aaa"
[] A character set (match any inside) [aeiou] matches any vowel

Example: [\w\.-]+ means: Match any word, dot, or hyphen, one or more times.

Decoding RegEx Symbols (Part 2)

Anchors and Quantifiers for precise matching:

Symbol Name Meaning Example
. Dot Matches any character (except newline) a.b matches "axb", "a2b"
\s Whitespace Matches spaces, tabs, or newlines \w\s\w matches "A B"
^ Caret Matches the start of a string ^Hello
$ Dollar Matches the end of a string end$
* Star Matches 0 or more repetitions hi* matches "h", "hi", "hiii"
? Question Matches 0 or 1 repetition (Optional) colors? matches "color" or "colors"

Pro Tip: Use ^ and $ together to ensure an entire string matches a pattern exactly (e.g., ^\d{6}$ for a 6-digit postal code).

Topic 2: Comprehensions

A concise way to create lists, dictionaries, or sets.


# List Comprehension
squares = [x**2 for x in range(5)] # [0, 1, 4, 9, 16]

# Dictionary Comprehension
prices = {'apple': 1.0, 'banana': 0.5}
taxes = {k: v * 1.1 for k, v in prices.items()}

Comprehensions (3 Examples)

  1. Filtering: Creating a list of only even numbers from 1 to 100.
  2. Transformation: Converting a list of strings to uppercase.
  3. Set Creation: Extracting unique first letters from a list of names.

Comprehension Code Gallery

1. Filtering (Even numbers 1-20)


# Traditional way: 4 lines. Comprehension: 1 line.
evens = [n for n in range(1, 21) if n % 2 == 0]

2. Transformation (Uppercase Names)


names = ["alice", "bob", "charlie"]
shouting = [name.upper() for name in names]
print(shouting) # ['ALICE', 'BOB', 'CHARLIE']

3. Set Creation (Unique Initials)


students = ["Alice", "Andrew", "Bob", "Beatrice", "Charlie"]
initials = {name[0] for name in students}
print(initials) # {'A', 'B', 'C'} - Note: Duplicates are removed!

Topic 3: Decorators

Functions that "wrap" other functions to modify their behavior without changing their source code.


def my_decorator(func):
    def wrapper():
        print("Something before.")
        func()
        print("Something after.")
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")

Decorators (3 Examples)

  1. Logging: Recording when a function starts and ends.
  2. Authorization: Checking if a user is logged in before running a function.
  3. Timing: Measuring how many seconds a function takes to execute.

Topic 3: Working with Dates and Time

The datetime module is part of the Python Standard Library.

  • date: For Year, Month, Day.
  • time: For Hours, Minutes, Seconds.
  • datetime: A combination of both.
  • timedelta: Used for date arithmetic (adding/subtracting time).

Parsing and Formatting

Converting strings to objects (Parsing) and objects to strings (Formatting).


from datetime import datetime

# Formatting (strftime: String From Time)
now = datetime.now()
print(now.strftime("%Y-%m-%d %H:%M")) # 2026-03-11 19:22

# Parsing (strptime: String Parse Time)
date_str = "12-25-2025"
dt_obj = datetime.strptime(date_str, "%m-%d-%Y")

Datetime in Practice

1. Age Calculator (Date Arithmetic)


from datetime import date
birth_date = date(2000, 5, 20)
today = date.today()
age = today.year - birth_date.year - ((today.month, today.day) < (birth_date.month, birth_date.day))
print(f"User is {age} years old.")

Deep Dive: One-Line Age Logic

age = today.year - birth.year - ((today.month, today.day) < (birth.month, birth.day))

Step 1: Year Difference

today.year - birth.year calculates the maximum age possible this year.

Step 2: Tuple Comparison

Python compares tuples item-by-item: (month, day).

  • If the current date is before the birthday, the expression is True.
  • If the birthday has passed, it is False.

Step 3: Boolean Arithmetic

In Python, True == 1 and False == 0.

  • If True (1): Subtract 1 from the age (birthday hasn't happened yet).
  • If False (0): Subtract 0 (age remains the same).

Born May 20: (3, 11) < (5, 20) is True (1).
