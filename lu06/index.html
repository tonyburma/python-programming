Introduction to Flask

Modern Web Apps & APIs with Python

1. How Modern Apps Work

Separation of Frontend and Backend.

  • Client: The UI (React, Swift, Android).
  • Server: The Logic (Flask, Node, Go).
  • API: The messenger using HTTP.

Example: Your phone asks for data (Request), the server sends JSON (Response).

2. What is Flask?

  • A Micro-framework for Python.
  • Extremely lightweight and easy to scale.
  • No "enforced" way to do things (Flexibility).
  • Built-in development server and debugger.

3. Installing Flask


# Create folder
mkdir my_flask_app && cd my_flask_app

# Set up Virtual Environment
python -m venv venv
source venv/bin/activate # Mac/Linux
venv\Scripts\activate    # Windows

# Install
pip install flask

4. Structure of Flask App


my_flask_app/
│
├── app.py           # Logic
├── templates/       # HTML files
└── static/          # CSS, JS, Images

5. Developing Home Page

Inside app.py:


from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "
Hello World!
"

if __name__ == "__main__":
    app.run(debug=True)

6. Adding Other Routes


@app.route("/about")
def about():
    return "This is the about page."

@app.route("/user/")
def profile(name):
    return f"Welcome, {name}!"

7. HTML Templates

Using render_template to serve HTML files.


from flask import render_template

@app.route("/")
def index():
    return render_template("index.html")

Flask looks for these in the /templates folder.

8. API Routes

Return a Dictionary to automatically serve JSON.


@app.route("/api/info")
def api_info():
    return {
        "framework": "Flask",
        "language": "Python",
        "version": 3.0
    }

9. Testing with Postman

  1. Run your app: python app.py
  2. Open Postman.
  3. Set method to GET.
  4. URL: http://127.0.0.1:5000/api/info
  5. Check the JSON output!
